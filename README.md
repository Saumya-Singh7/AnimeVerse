# AnimeVerse

# Overview
AnimeVerse is a web application that allows users to explore anime characters, search by name, and view detailed information. It’s designed to be beginner‑friendly yet scalable, showcasing skills in HTML, CSS, JavaScript, and Generative AI integration.

# Features
🔍 Search Functionality: Find anime characters quickly by name.

🎨 UI/UX Design: Clean, centered layout with the “AnimeVerse” heading above the search bar.

📂 Scalable Structure: Organized folder system for easy contributions.

🤖 Future Scope: Integration with AI for recommendations and character insights.
